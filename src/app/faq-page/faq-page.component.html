DÚVIDAS FREQUENTES (FAQ)

SOBRE O SISTEMA

1.1 QUEM PODE USAR ESTE SISTEMA?

Este sistema é exclusivo para instituições de saúde, como hospitais, clínicas, farmácias e postos de saúde

1.2 O SISTEMA FUNCIONA ONLINE?

Sim. Ele é totalmente baseado na web, acessível de qualquer dispositivo com internet e credenciais autorizadas.

GESTÃO DE MEDICAMENTOS

2.1 É POSSÍVEL CADASTRAR TODOS OS TIPOS DE MEDICAMENTOS?

Sim, o sistema permite o cadastro detalhado dos medicamentos, incluindo lote, validade, fabricante e dosagem.

2.2 O SISTEMA ALERTA SOBRE MEDICAMENTOS VENCIDOS?

Sim. Há notificações automáticas para medicamentos próximos do vencimento.

CONTROLE DE ESTOQUE

3.1 O SISTEMA CONTROLA ENTRADAS E SAÍDAS DE ESTOQUE?

Perfeitamente. Toda movimentação de entrada (compra, recebimento) e saída (distribuição, consumo) é registrada.

3.2 EXISTE RELATÓRIO DE ESTOQUE MÍNIMO?

Sim. O sistema gera alertas quando algum medicamento atinge o estoque mínimo configurado.

SEGURANÇA E ACESSO

4.1 COMO FUNCIONA O CONTROLE DE USUÁRIOS?

O acesso é restrito e por níveis. Cada usuário tem permissões específicas, garantindo segurança na gestão.

4.2 O SISTEMA ARMAZENA DADOS SENSÍVEIS?

Sim, de forma segura e criptografada, seguindo boas práticas de proteção de dados.

SUPORTE TÉCNICO

5.1 E SE EU TIVER PROBLEMAS TÉCNICOS?

Nosso suporte está disponível em horário comercial para resolver qualquer problema ou dúvida.

5.2 O SISTEMA RECEBE ATUALIZAÇÕES?

Sim. Realizamos atualizações periódicas para melhorias, correções e novas funcionalidades.
